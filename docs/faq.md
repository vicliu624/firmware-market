# FAQ

## Why is there no upload button?

This is a GitHub-driven index. Firmware binaries live in GitHub Releases; manifests are submitted via PR.

## Why are community builds hidden by default?

To reduce risk for non-expert users. You can opt in to view community builds.

## Does this site vouch for third-party firmware?

No. Trust tiers are displayed, and users are responsible for verifying compatibility.

## Does Web Flash work everywhere?

No. WebSerial is supported only in desktop Chrome/Edge. Other browsers fall back to download + CLI.
